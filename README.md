## HTML

### HTML 基础语法

#### 什么是 HTML？

HTML（HyperText Markup Language，超文本标记语言）是一种*标记语言*，它告诉 Web 浏览器如何组织网页。

HTML 由一系列的**元素**组成，你使用这些元素来包围、包装或*标记*内容的不同部分，使其以某种方式显示或运作。

HTML 存在于扩展名为 `.html` 的文本文件中，这种文件被称为 **HTML 文档**，或简称**文档**。

你将遇到的最常见的 HTML 文件是 `index.html`，它通常用于包含网站主页的内容。同样常见的是看到带有各自 `index.html` 的子文件夹，因此一个网站可以在不同的位置有多个索引文件。

> [!TIP]
> 
> HTML 中的标签不区分大小写。但是，为了保持一致性和可读性，<span style="color: green;">最佳实践是所有标签都用小写书写</span>。

#### 剖析一个 HTML 元素

元素是**开始标签**、**内容**和**结束标签**的总和。
